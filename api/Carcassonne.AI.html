    The AIDecisionRequester sets up the allowed number of actions for the AI and and requests a decision when needed. Version 1.0

    AIWrapper

    The AIWrapper acts as a middle-man between the AIPlayer-class and the data it needs and actions it can perform. It separates the AI logic from the code implementation. Its specific purpose is to allow the exact same AIPlayer-class to be used in the real environment and the training environment. This means the AIWrapper class will look different in both these project, as the code running the game differs in the two implementations. Version 1.0
